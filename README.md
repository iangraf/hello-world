# vue-test

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Runs locally in node
```
npm run build
node app.js
```

### Runs local node static file server
```
npm install -g serve
npm run build
serve -s dist
```

### Lints and fixes files
```
npm run lint
```
